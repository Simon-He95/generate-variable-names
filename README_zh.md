生成多种变量名, 功能类似于CodeIf, 实现了一个在vscode中直接操作的插件

![demo](/assets/demo.gif)

## 使用
通过快捷键 ***Mac***: `cmd` + `shift` + `r` | ***Windows***: `ctrl` + `shift` + `r` 启动命令，如果你选中了中文变量名则直接列出变量名列表，如果未选择，则需要再多一步输入中文含义的变量名，再列出结果
